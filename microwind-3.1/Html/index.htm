Getting Started

To get your MICROWIND3 program started (Version 3.1), download the latest version of the software at the web site www.microwind.org or insert the MICROWIND3 CD-Rom into the CD player. The software runs on Windows 95, 98, 2000, NT, XPoperating systems.

  1. Unzip the file in your personal directory (Like "Microwind3" in "My Documents")
  2. Double click the Microwind3 icon in the sub-directory "system" to start the software . The default design rule for version 3.1 is "cmos90nm.RUL".

The command line may include two parameters: The First parameter is the default mask file loaded at initialization. The second parameter is the design rule file loaded at initialization. Example : The command " microwind3 fadd cmos018.rul " executes MICROWIND3 with a default mask file " fadd.MSK " and the rule file " cmos018.RUL ". You may program the Microwind3 icon by a click with the right button, then "properties".
